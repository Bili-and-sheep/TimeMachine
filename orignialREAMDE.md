# Welcome to the Time Machine!



This is motiveted by a School requirement, but I mostly du to my love for Apple and the nonexistent history of Apple product (mosly accessories) in the market, I decided to create a Time Machine that can travel back in time to retrieve Apple products that are no longer available.


- Same look as the 2017 Apple Store
- Community driven: Users can request specific Apple products to be retrieved from the past.
- Community can also contribute by Creating a Product page (A manuel review will be done before the product page is published to ensure the quality of the content)
-


- Product Page :
    - Type Of Product (iPhone/MacBook/Apple Watch/Accessory/...)
    - Product Name
    - Release Date
    - Year of Discontinuation
    - OS release with (iOS/macOS/watchOS/...)
    - Last OS version supported
    - Description
    - Image
    - Options (Colors/Sotackge/Size/...)
    - Price (input by user)
    - Price with inflation (calculated based on release date and current date)
    - Tags (input by user to categorize products)
    - Masked Sources use

Web Site body :
- Home Page: Showcasing featured products and latest product added.
- Apple .com Felling like, same iPhone/MacBook/Apple Watch/Accessory
- Search Functionality: Allow users to search for products based on name, type, release date, etc.
- User Account: Allow users to create an account (randomize ID to ensure maxime security) to contribute to the community.
- Admin Panel: For manual review of product pages and managing user contributions.
- Contact Page: Allow users to contact the team (me and any futur member)for support or inquiries.
- About Page: Information about the project, its goals, and the team behind it.
